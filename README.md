# BasePlay

Sports prediction market on Base blockchain.

## Features

- Create prediction pools for sports events
- Bet on Home/Away/Draw outcomes
- Claim winnings after pool resolution
- Owner-controlled pool management

## Tech Stack

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Wagmi v2
- Viem v2
- Base Mainnet

## Contract

Address: `0x53420899c0b98105e0fe4ab5ae09033b5ef5880d`

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Deploy

```bash
npm run build
vercel --prod
```
